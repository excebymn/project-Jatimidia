# Dokumentasi Proyek

## Latar Belakang
Proyek ini dibuat sebagai bagian dari kebutuhan lomba/penilaian yang melibatkan pengembangan tampilan web statis. Struktur file dibuat agar mudah dibuka dari berbagai lingkungan.

## Manfaat
- Mempermudah pengguna memahami struktur proyek.
- Menyediakan panduan penggunaan.
- Memberikan dokumentasi resmi dalam format PDF dan README.

## Cara Menggunakan
1. Ekstrak semua file proyek ke dalam satu folder.
2. Jika membuka dari **PC**, gunakan ekstensi **Live Server** di Visual Studio Code agar seluruh file berjalan optimal.
3. Buka file `index.html` melalui Live Server.
4. Jika ingin melihat file dari GitHub Pages, cukup unggah folder ke repository GitHub.
